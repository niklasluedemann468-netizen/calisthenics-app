Calisthenics Coach iPhone PWA – mit den Grafiken aus deinem Trainingsplan

So nutzt du sie:
1. ZIP entpacken.
2. Alle Dateien zu GitHub Pages hochladen.
3. Link in Safari auf dem iPhone öffnen.
4. Teilen → Zum Home-Bildschirm.

Die Übungsbilder wurden aus deinem hochgeladenen Trainingsplan ausgeschnitten und in die App eingebaut.
