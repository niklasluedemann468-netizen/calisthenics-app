Calisthenics Coach PWA für iPhone

Dateien bei GitHub Pages hochladen:
- index.html
- style.css
- app.js
- manifest.json
- sw.js
- icon-192.png
- icon-512.png

Danach auf dem iPhone:
1. GitHub-Pages-Link in Safari öffnen.
2. Teilen-Symbol tippen.
3. „Zum Home-Bildschirm“ wählen.
4. App starten.

Die App funktioniert danach wie eine Web-App mit Icon, Vollbildmodus und Offline-Cache.
